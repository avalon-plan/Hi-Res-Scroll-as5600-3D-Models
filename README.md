不想用我的底座，可自己修改
先说说我这个VER3.0版本的问题
1. 径向磁铁用的直径6高2.5mm，我使用了嘉立创的3301PA   白色，磁铁塞入上盖有点不够紧，因此我使用了美纹贴纸贴住了，如果你也是此模型最好修改调小一点，但是一定不要去刻意扩孔（扩孔会导致磁体中心对不准as5600模块上芯片的中心），如果是CNC的话个人觉得没有必要改，也可以不修改此孔

3. 底座限制电池802540的长边加强筋没有考虑到因为电池的保护电路板实际比40mm更长一点，导致我第一次塞入的时候给加强筋直接干断了
4. 底座多的一个顶住开发板的加强筋那一块感觉有点大了，导致焊接电源线的时候，有点卡手，最好把这个加强筋设计的再小一点
5. 底盖是有三个加强筋，一个是卡住侧边开孔的按钮，两个是顶住开发板正面的D2、D3和D10、D9、D8，因此D2、D3、D10、D9、D8不要上锡或者焊接东西。焊接请使用D0 D1 D4 D5 D6 D7 GND 3.3V 5V这几个接口

以上问题我有时间会修改下，打印下一个外壳
